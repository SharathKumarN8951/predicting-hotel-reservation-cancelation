# Hotel Reservation Cancellation Prediction

## About the Project

This project focuses on predicting whether a hotel reservation is likely to be canceled using machine learning techniques. The aim is to help hotels identify potential cancellations in advance so they can improve room management and reduce revenue loss.

During this project, I worked on understanding the dataset, cleaning the data, exploring different features, and building a prediction model. I also learned how different customer and booking details can influence reservation cancellations.

---

## Project Files

- `predicting hotel reservation cancelation.ipynb` – Jupyter Notebook containing the complete project.

## Project Workflow

The project was completed in the following steps:

- Loaded the hotel reservation dataset.
- Cleaned and prepared the data for analysis.
- Performed exploratory data analysis (EDA).
- Converted categorical data into numerical values.
- Split the dataset into training and testing sets.
- Trained a machine learning model.
- Evaluated the model using different performance metrics.
- Predicted whether a reservation would be canceled.

---

## Features Used

Some of the important features in the dataset include:

- Number of adults
- Number of children
- Number of weekends stayed
- Number of weekdays stayed
- Meal plan
- Room type reserved
- Lead time
- Arrival month
- Market segment
- Previous cancellations
- Average room price
- Special requests

---

## Model Evaluation

The model performance was evaluated using:

- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1 Score

These metrics helped measure how well the model predicts reservation cancellations.
